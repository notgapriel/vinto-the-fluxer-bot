# Support

## Where To Ask

- Realtime support server (Vinto): https://fluxer.gg/qDoq4Tf0
- Usage questions: open a GitHub Discussion or Issue.
- Confirmed bugs: open a Bug Report issue using the template.
- Security concerns: follow [SECURITY.md](SECURITY.md) and report privately.

## Before Opening an Issue

- Check [README.md](README.md) and [docs/CONFIGURATION.md](docs/CONFIGURATION.md).
- Search existing issues/discussions for duplicates.
- Test with the latest `main` branch and current dependencies.
- For self-hosting issues, mention whether you changed the default Fluxer endpoint settings.

## What To Include

- Node version and OS
- relevant `.env` flags (redact secrets)
- whether you use the default Fluxer endpoints or changed `API_BASE` / `GATEWAY_URL`
- exact command/query that failed
- logs or stack traces
- expected behavior and actual behavior
